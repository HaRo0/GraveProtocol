To start you need to get a TD Gateway, which can be crafted in a normal Workbench.

Once you enter the Portal you will see a Fox NPC, interacting with it opens a shop in which you should buy the Free options.

Next to the Fox are borken statues, those are Tower bases, which can be interacted with to create a Tower.

Once you start the game using the Fox NPC, you get the needed currency to buy Towers, which you should do to stop the enemies from killing the Fox npc and you losing.

If you clear all waves you get to the next level, if you clear all levels you can prestige at the Fox npc and start over again with increased stats.
